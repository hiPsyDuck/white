# white

typecho 主题 白

下载解压后将文件夹改名为`white`， 放入/usr/themes，进入后台启用。

添加独立页面`bb.html` `archive.html` `links.html` `about.html` 分别对应底部导航栏的 `留言` `归档` `邻居` `关于`
